Weather App

Objective - To develop a graphical application that fetches and displays real-time weather information for a user-specified location.

Steps Performed - 
1.Designed a graphical user interface using PyQt.
2.Added an input field to allow users to enter a city or location name.
3.Integrated a weather data service to fetch current weather details.
4.Implemented logic to retrieve and process live weather data.
5.Displayed essential weather information such as temperature, humidity, and weather conditions.
6.Handled basic input validation and error scenarios to ensure smooth user interaction.

Tools Used - Python,PyQt

Outcome - A simple and intuitive GUI-based Weather Application that allows users to quickly check current weather conditions for any specified location in an easy-to-use interface.
